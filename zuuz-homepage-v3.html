FAQ

Common questions.

Anything else — our team responds within 24 hours.

How quickly can ZUUZ go live?+
For most enterprise environments, the timeline from IT API approval to first agents live on real data is 2–3 weeks. Week 1: security review and API approvals. Week 2: integration configuration and agent setup. Week 3: agents running on live data with measurable time savings. Your stack stays exactly where it is — no migration, no data movement.
Do I have to deploy the entire platform at once?+
No. The recommended path is one department, one use case. The platform infrastructure (integrations, RBAC, security) is deployed once. The first use case runs on that infrastructure. Adding a second department uses the same integrations — days, not months.
How is ZUUZ priced?+
Three layers: Platform Foundation (fixed monthly), Department Modules (flat fee per activated department), and Work Credits (charged only on successful task completion — never for failed or incomplete tasks). No per-seat pricing. Ever. See our full pricing page.
Is this replacing our existing tools?+
No. ZUUZ connects to your existing stack — SAP, Oracle, Salesforce, Teams, BambooHR, Zendesk — via API. Your tools stay where they are. ZUUZ adds an AI intelligence and automation layer on top. No rip-and-replace. No data migration.
What does IT have to do?+
IT reviews our SOC 2 documentation and architecture diagrams, then approves the API connection credentials for each connected system. They approve once — they're not involved in ongoing operation or when new use cases are added using the same connections.
What if the AI makes a mistake?+
ZUUZ agents act on defined, structured outputs — they do not make judgment calls in ambiguous situations. Every agent action is logged in the A2A audit trail. High-stakes decisions (like deal approvals) surface AI-assembled context to the human decision-maker — the AI informs, the human decides.
